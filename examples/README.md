## How to view

Just open the `.html` file in any browser. All CDN libs load automatically.

## How to fork an example

```bash
# 1. Copy an example into your output
cp examples/proposal-saas-startup.html output/my-proposal.html

# 2. Edit it manually for your client
# Or, regenerate with the CLI:
atk gen proposal --type=seo-diagnostic --client="My Client"
```
